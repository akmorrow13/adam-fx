# adam-fx

A Scala library extending [ADAM](https://github.com/bigdatagenomics/adam) and [BDG-formats](https://github.com/bigdatagenomics/bdg-formats) 
to load .vcf files annotated with [SnpEff](http://snpeff.sourceforge.net/).
